🧠 Primer contacto con Python

En este primer contacto con Python, lo más fácil fue entender cómo usar print() para mostrar información en pantalla y declarar variables para guardar datos. La sintaxis del lenguaje es bastante clara y fácil de leer, lo que facilita el aprendizaje cuando uno recién comienza a programar. 💻

Por otro lado, lo más desafiante fue acostumbrarme a la forma correcta de escribir el código y evitar pequeños errores de sintaxis, como usar signos incorrectos o escribir mal alguna instrucción. ⚠️
También resultó un poco complejo comprender cómo funcionan algunos módulos, como math, y cómo utilizar sus funciones dentro del programa.

Sin embargo, con práctica y revisando los errores que aparecen, poco a poco se vuelve más fácil comprender cómo funciona Python. 🚀
Esto hace que el proceso de aprendizaje sea progresivo y cada vez más interesante.
